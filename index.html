About

Originally from North-East Ohio, I moved to NYC in 2020. When COVID hit I began the process of transitioning my career from hospitality management to coding. In my spare time I enjoy reading tech blogs, gaming and spending time with my fiance and our dog.

I'm an inquisitive person, always learning new things, meeting interesting people and looking for new perspective. I'm always on tinkering with a new project and can be found most nights at my computer, wrapping my head around new concepts and features.

Around the Web

Veterinary Clinic

This was my phase 4 project at Flatiron School. Along with a partner I build a web application for a fictional Veterinary Clinic that allows a user create an account, register their pet and schedule an appointment. This project uses a React frontend with a Ruby on Rails backend.

Squad Builder

This was my phase 3 project at Flatiron School, the front end is React and the backend is Ruby with Sinatra. Squad Builder allows a user to search its database of games for other users looking to create a "squad" for their gaming session.

Etch-a-Sketch

I'm not choosing to display the Etch-A-Sketch for its complexity but rather the number of important skills it taught me. This small project was one of the first things I built and because of it I have a deep understanding of Flexbox. This project allows you to draw in the browser as well as change the size of the board, color of the marker and switch to an eraser or even perform a full reset.

Civic Doody

My capstone project at Flatiron School. I used Google Maps API and Google Geolocator API on Google Cloud Platorms to dynamically render, add and delete locations based on user input. I handled user authentication via session cookies and password encryption with the bcrypt gem. Users were able to communicate via comments on the rendered pins. This project used a React frontend and Ruby on Rails backend.
